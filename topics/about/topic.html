About

Title:
Your Own Web (YOW) Free Sample[1]
Author:
Scott Haines[1]
Description:
Your Own Web Free Sample provides simple browser pages in a Git version control repository. Start with YOW Free Sample. Change its pages, graphics and images to create and maintain your own browser pages. Use a version control repository to track your changes.
YOW Free Sample is in the public domain to minimize restrictions on its use.
You can use free open source software to edit and maintain your own browser pages. I recommend the free open source software Amaya, Git and Tortoise Git. For the Outline page I suggest BlueGriffon rather than Amaya.
Version:
2.2.0
Release level:
production release
Reason for this release level:
YOW Free Sample 2.2.0 works.
