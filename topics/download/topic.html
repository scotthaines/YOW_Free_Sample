Download

Installer

Install YOW Free Sample pages and its Git repository. The Git version control program for Windows is also installed if it is not already installed. All of these are on your computer for your use.

Zip file

Copy the YOW Free Sample pages from the zip file. The zip file does not have the Git repository. Git version control for Windows is not installed on your computer.

Clone

Clone the repository to your computer. To do this you need the Git version control program on your computer and you need to know how to use it.

  • Download zip file v1.6.0.zip
